# Huajie Su

This repository hosts my personal academic homepage built with Hugo/Wowchemy and deployed with GitHub Pages.

Website: https://wangsu-dev.github.io/

## Editing notes

Main content files:

- `content/authors/admin/_index.md` — biography and research interests
- `content/project/example/index.md` — research overview
- `content/publication/example/index.md` — publication entry
- `config/_default/config.yaml` — site title and base URL
- `config/_default/params.yaml` — site metadata and contact settings

Deployment is handled by `.github/workflows/hugo.yaml`.
